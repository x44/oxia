# create-oxia

## Scaffolding for Oxia projects

With npm
```bash
npm create oxia@latest
```
With pnpm
```bash
pnpm create oxia
```
With yarn
```bash
yarn create oxia
```

<br/>
You can also specify a project name and a template name:


```bash
# npm
npm create oxia@latest my-project minimal

# pnpm
pnpm create oxia my-project minimal

# yarn
yarn create oxia my-project minimal
```
